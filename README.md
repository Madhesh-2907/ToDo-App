# TaskFlow | Todo App

A simple client-side todo list web app built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Delete individual tasks
- Clear all tasks
- Live counters for total tasks and completed tasks

## Files

- `index.html` - app structure and markup
- `style.css` - visual styling and layout
- `script.js` - task behavior and interaction logic

## Usage

1. Open `index.html` in a web browser.
2. Type a task in the input field.
3. Click `Add Task` or press `Enter`.
4. Use the checkmark button to mark tasks complete.
5. Use the trash button to remove tasks.
6. Click `Clear All Tasks` to reset the list.

## Notes

- This app is fully client-side and does not persist tasks after page refresh.
- No build or installation is required.
